## Переменные среды для AI агента [BotHub](https://bothub.ru)
Так как для проекта не было возможности использовать [DeepSeek], решили использовать промежуточный сервис [Bothub](https://bothub.ru)
Токен можно получить по ссылке https://bothub.ru/profile/for-developers

| Переменная среды    | Обязательность | Тип данных        | Описание                  |
|---------------------|----------------|-------------------|---------------------------|
| AI__API_KEY         | Обязательно    | Строка            | Секретный ключ            |
| AI__MAX_CONNECTIONS | Опционально    | Целое число (> 0) | Количество подключений    |
| AI__MODEL           | Опционально    | Строка            | Имя модели                |
| AI__TIMEOUT         | Опционально    | Целое число (> 0) | Время ожидания в секундах |
| AI__BASE_URL        | Опционально    | Строка            | Базовый URL (сервиса)     |


## Переменные среды для [Unsplash](https://unsplash.com/)
Токены можно получить по ссылке https://unsplash.com/oauth/applications

| Переменная среды              | Обязательность | Тип данных         | Описание                    |
|-------------------------------|----------------|--------------------|-----------------------------|
| UNSPLASH__ID                  | Обязательно    | Целое число        | Идентификатор приложения    |
| UNSPLASH__ACCESS_KEY          | Обязательно    | Строка             | Ключ доступа                |
| UNSPLASH__SECRET_KEY          | Обязательно    | Строка             | Секретный ключ              |
| UNSPLASH__MAX_CONNECTIONS     | Опционально    | Целое число (> 0)  | Количество подключений      |
| UNSPLASH__TIMEOUT             | Опционально    | Целое число (> 0)  | Время ожидания в секундах   |
